📝 HTML Basics

What HTML Tags Look Like:

<tag>content</tag>

Essential Tags:

<h1>...</h1>
Headings (h1 = biggest, h6 = smallest)
<p>...</p>
Paragraph of text
<a href="url">link text</a>
Link to another page
<img src="file.jpg" alt="description">
Image (note: no closing tag!)
<ul>
  <li>item</li>
</ul>
Bullet list (unordered)
<ol>
  <li>item</li>
</ol>
Numbered list (ordered)
<div>...</div>
Container/box for grouping content
<strong>...</strong>
Bold text
<em>...</em>
Italic text

🎨 CSS Basics

How CSS Works:

selector { property: value; }

Common Properties:

color: purple;
Changes text color
background-color: pink;
Changes background color
font-size: 20px;
Changes text size
padding: 15px;
Space inside an element
margin: 15px;
Space outside an element
border: 2px solid purple;
Border around element
border-radius: 8px;
Rounded corners
text-align: center;
Align: left, center, or right

💡 Helpful Tips

  • Always save your file before refreshing the browser
  • Close every tag you open: <h1>...</h1>
  • CSS goes in the .css file, HTML goes in the .html file
  • Use Ctrl+S (or Cmd+S on Mac) to save
  • If something looks wrong, check for typos first
  • Make sure all semicolons are in your CSS
  • You can right-click and choose "View Page Source" to see HTML
  • The browser's developer tools (F12) can help you debug

🔧 Quick Fixes

"My page is blank" Check if the HTML file is saved. Then refresh the browser (Ctrl+R or Cmd+R).
"My CSS isn't working" Make sure <link rel="stylesheet" href="style.css"> is in your <head> tag and the filename matches.
"My image won't show" Check that the filename and path match exactly. File names are case-sensitive (.jpg vs .JPG matters!).
"Colors aren't changing" Make sure you saved the CSS file too. Close and reopen the HTML file in your browser.
"Tags are showing on the page" You might have forgotten to close a tag or have a typo. Check that all tags are spelled correctly and closed.
