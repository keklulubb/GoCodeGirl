# GCG-webdev

Workshop materials for **GoCodeGirl** — an intro to web development workshop for girls and non-binary youth (Grades 7-10).

## What's in this repo

- **`workshop_code/`** — Starter code students work with during the workshop. Open `index.html` to get started.
- **`cheatsheet.html`** — HTML & CSS quick reference. Open in a browser tab during coding.
- **`slides/`** — Workshop presentation slides.
- **`reference/`** — Completed example and original skeleton code for facilitator reference.

## Workshop Overview

**Duration:** 4 hours (including lunch and breaks)  
**Audience:** 25 students, Grades 7-10  
**Topic:** Building a personal website with HTML & CSS

### Schedule

| Time | Activity |
|------|----------|
| 10:00 | Welcome & Intro |
| 10:20 | Session 1: Build your core site |
| 11:30 | Lunch |
| 12:15 | Session 2: Customization & Challenges |
| 1:20 | Show & Tell |
| 1:40 | Wrap-up |

### Session 1 (Core)
Students build a personal homepage (`index.html`) and fun facts page (`funFacts.html`), learning HTML structure and CSS styling through color customization.

### Session 2 (Challenges)
Students choose from:
- **Level 1:** Photo collage page (`collage.html`)
- **Level 2:** Personality quiz (`quiz/`)
- **Level 3:** Advanced CSS (gradients, animations, fonts)
- **Level 4:** Override CSS variables for individual elements

## How to Run

No build tools needed — just open any `.html` file in a browser.

For live reloading during development:
- **VS Code:** Install "Live Server" extension, right-click an HTML file → Open with Live Server
- **Terminal:** `npx serve workshop_code`

## Tools

- **Code Editor:** VS Code (recommended) or any browser-based editor (Replit, CodePen)
- **Deployment:** [Netlify Drop](https://app.netlify.com/drop) — drag and drop your folder to get a live URL
