HOW TO ADD YOUR PHOTOS

1. Save your image files to this folder
   Example names: profile.jpg, photo1.jpg, sunset.png

2. In your HTML, reference them like this:
   <img src="images/profile.jpg" alt="description">

3. Or use images from the web by copying the full URL:
   <img src="https://example.com/photo.jpg" alt="description">

Tips:
- Use .jpg, .png, or .gif files
- Keep file names simple (no spaces or special characters)
- Always include an alt= description for accessibility
